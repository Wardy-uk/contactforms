Popup widget demo

Contact form popup widget, powered by BriefYourMarket.

This page demonstrates the contact form loading as a popup widget. The dialog is configured to appear on visit number 5 using the parameter afterVisits=5.

  • The form itself is injected by GetContactForm.
  • The popup behaviour is controlled by DisplayContactForm.
  • Popup will persist forever (cookie expiry = 0 days).
